# Updating document.title with useEffect

This repo is a quick example of a problem that can be solved with `useEffect`.

To run locally on your machine, download the code, and then run:

- `npm install`
- `npm run start`

If you're a student of “The Joy of React”, you can learn more about running code locally in the [“Tools of the Trade”](https://courses.joshwcomeau.com/joy-of-react/11-tools-of-the-trade) module.
